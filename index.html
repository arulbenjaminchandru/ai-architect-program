3-Month Program

The Curriculum

24 live weekend sessions across 3 months. Every Saturday is concept + demo. Every Sunday is hands-on lab. Weekdays are guided coursework.

Month 1 — AI Foundations
From zero to confidently working with AI APIs and building real prompts
Weekend 1 · Apr 18–19 How AI & LLMs Work +
Saturday · Concept
What is an LLM? How does AI think? Tokens, training, prediction — no mathematics. Key parameters: temperature, context window, models.
Sunday · Lab
Your first Claude API call — send a prompt, read the response, inspect token usage. Build a working script from scratch.
Weekday Coursework
Free Udemy: Python & Gen AI Made Simple — complete Module 1 (Python basics). Practice notebook: Python setup & API basics.
Weekend 2 · Apr 26–27 Prompt Engineering +
Saturday · Concept
Anatomy of a prompt: system message, user role, assistant response. Techniques: few-shot, chain-of-thought, structured JSON output.
Sunday · Lab
Build a document summariser that outputs a structured JSON summary using Claude and XML-tagged prompts.
Weekday Coursework
Free Udemy: Gen AI foundations module. Practice notebook: prompt iteration — write 3 versions, compare outputs.
Weekend 3 · May 3–4 Tool Use — Giving AI Hands +
Saturday · Concept
What is tool use? How the AI decides to call an external function. Tool call loop: AI calls → you run → return result → AI continues.
Sunday · Lab
Build a 2-tool assistant: calculator + weather lookup using Claude. Watch it choose the right tool automatically.
Weekday Coursework
Coursework: Design a tool schema for a repetitive task in your own domain. Practice notebook: tool call exercises.
Weekend 4 · May 10–11 Memory & Conversations +
Saturday · Concept
Why AI forgets everything — and how to fix it. Stateless vs stateful AI. Sliding windows and prompt caching explained.
Sunday · Lab
Build a multi-turn chatbot with a sliding memory window. Add a summarisation step that compresses old history automatically.
Weekday Coursework
Coursework: Extend chatbot to compress old history when the window gets full. Practice notebook: memory patterns.
Month 2 — Building Real AI Systems
Go from API calls to complete, working applications with real data
Weekend 5 · May 17–18 Embeddings & Semantic Search +
Saturday · Concept
What are embeddings? How AI finds meaning in text — not just keywords. Cosine similarity in plain terms. Vector databases overview.
Sunday · Lab
Build a semantic search engine over 50 documents using ChromaDB. Run 5 queries and compare keyword vs semantic results.
Weekday Coursework
Practice notebook: embedding exercises + compare search methods on your own dataset.
Weekend 6 · May 24–25 RAG — Connect AI to Your Data +
Saturday · Concept
RAG pipeline: chunk → embed → store → retrieve → generate. Chunking strategies and grounding — Claude answers only from your data.
Sunday · Lab
Build a PDF Q&A app: upload any document, ask Claude questions, get grounded answers. Test with 3 chunk sizes.
Weekday Coursework
Practice notebook: RAG exercises + course material review for Week 6.
Weekend 7 · May 31–Jun 1 AI Project Structure & Claude Code +
Saturday · Concept
CLAUDE.md — the config file that makes Claude understand your project. Project context, conventions, constraints, custom slash commands.
Sunday · Lab
Write a production CLAUDE.md for the RAG project. Test 3 custom slash commands. Claude Code overview as autonomous coding agent.
Weekday Coursework
Coursework: Write a CLAUDE.md for your Weekend 6 project. Share in class next session.
Weekend 8 · Jun 7–8 AI Agents — AI That Acts +
Saturday · Concept
Agent loop: perceive → think → act → observe → repeat. Planning patterns. When agents go wrong — loops, wrong calls, context drift.
Sunday · Lab
Build a goal-driven Claude agent that plans steps and uses tools to complete a multi-step task autonomously.
Weekday Coursework
Coursework: Give the agent 3 different goals — observe how it behaves, document failures and fixes.
Month 3 — Advanced Systems & Career Launch
Production-grade engineering + real job preparation and capstone delivery
Weekend 9 · Jun 14–15 Multi-Agent Systems +
Saturday · Concept
Why teams of agents beat one agent. Sequential, parallel, supervisor patterns. How agents communicate via tool outputs.
Sunday · Lab
Build a 3-agent pipeline: Researcher → Analyst → Writer, all powered by Claude. Each agent is specialised.
Weekday Coursework
Coursework: Add a Reviewer agent that flags errors before final output. Practice notebook: agent communication patterns.
Weekend 10 · Jun 21–22 MCP — AI Connected to Everything +
Saturday · Concept
MCP: the universal standard for connecting AI to any tool or data source. Architecture: Host ↔ Client ↔ Server. Tools, Resources, Prompts.
Sunday · Lab
Build a custom MCP server — expose a database query tool and file export resource. Connect to Claude Desktop and test live.
Weekday Coursework
Practice notebook: MCP exercises + course material review. List 5 systems in your work an MCP server could expose.
Weekend 11 · Jun 28–29 Safety, Evaluation & Deployment +
Saturday · Concept
AI Safety: prompt injection, output guardrails, responsible deployment. Evaluation: RAGAS metrics, LLM-as-judge, tracing.
Sunday · Lab
Run RAGAS evaluation on RAG app. Add guardrails. Deploy as FastAPI + Docker to a cloud free tier — get a live URL.
Weekday Coursework
Coursework: Run evaluation on your RAG app — score and fix the weakest dimension before capstone.
Weekend 12 · Jul 5–6 Capstone — Ship Your AI Product +
Saturday · Build Sprint
Architecture review + guided build sprint. Choose template: RAG app / Agent system / MCP-powered tool. Instructor guidance throughout.
Sunday · Demo Day
Live demo day — present your deployed AI product to the batch. Peer code review, certificate of completion.
Final Submission
Publish to GitHub with README, architecture diagram, and CLAUDE.md. Career roadmap: CCAF certification path and job strategy.
